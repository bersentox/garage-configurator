# AGENTS.md

## Проект
Этот репозиторий содержит несколько старых страниц и экспериментов, но активный проект конфигуратора находится только в папке:

garage-configurator-new/

Все изменения конфигуратора должны выполняться только внутри этой папки.

---

## Точка входа приложения

Единственная точка входа для проверки конфигуратора:

garage-configurator-new/index.html

Нельзя использовать для проверки:

- корневой index.html
- любые старые HTML страницы
- hero-scene.html напрямую

hero-scene.html является fragment-файлом и должен загружаться только через index.html.

---

## Файлы hero сцены

Hero сцена конфигуратора состоит только из следующих файлов:

garage-configurator-new/hero-scene.html  
garage-configurator-new/hero-scene.css  
garage-configurator-new/hero-scene.js  

Изменения hero должны выполняться только в этих файлах.

---

## Ограничения

Codex не должен:

- использовать старые версии конфигураторов
- изменять файлы вне garage-configurator-new без явного указания
- запускать hero-scene.html как отдельную страницу
- использовать корневой index.html

---

## Проверка изменений

Все изменения должны проверяться через:

garage-configurator-new/index.html

Пример проверки через локальный сервер:

python -m http.server 4173

http://127.0.0.1:4173/garage-configurator-new/

---

## Цель

Конфигуратор должен работать как изолированное приложение внутри папки:

garage-configurator-new/
